策略概览¶

Optopsy 包含 28 种内置期权策略,涵盖单腿、多腿和基于时间的策略。

策略分类¶

单腿策略 (4)¶

使用 call 或 put 的基本方向性交易。

  • Long Calls - 看涨方向性押注
  • Short Calls - 看跌或中性收益
  • Long Puts - 看跌方向性押注
  • Short Puts - 看涨收益策略

了解更多 →

Straddles & Strangles (4)¶

基于波动率的策略,结合 call 和 put。

  • Long Straddle - 从大幅波动中获利
  • Short Straddle - 从低波动率获得收益
  • Long Strangle - 成本更低的波动率交易
  • Short Strangle - 更宽的盈利区间收益

了解更多 →

垂直价差 (4)¶

有限风险的方向性策略。

  • Long Call Spread - 看涨借记价差
  • Short Call Spread - 看跌信用价差
  • Long Put Spread - 看跌借记价差
  • Short Put Spread - 看涨信用价差

了解更多 →

蝶式价差 (4)¶

针对特定价格区间的中性策略。

  • Long Call Butterfly - 在中间行权价获利
  • Short Call Butterfly - 从价格移动获利
  • Long Put Butterfly - 看跌蝶式
  • Short Put Butterfly - 反向 put 蝶式

了解更多 →

Iron 策略 (4)¶

具有确定风险的四腿策略。

  • Iron Condor - 区间震荡收益
  • Reverse Iron Condor - 突破交易
  • Iron Butterfly - 更紧的盈利区间
  • Reverse Iron Butterfly - Long 波动率

备兑策略 (2)¶

股票 + 期权组合(合成)。

  • Covered Call - 持股收益
  • Protective Put - 下行保护

日历价差 (4)¶

相同行权价,不同到期日。

  • Long Call Calendar - 时间衰减交易
  • Short Call Calendar - 反向日历
  • Long Put Calendar - 看跌时间价差
  • Short Put Calendar - 反向 put 日历

了解更多 →

对角价差 (4)¶

不同行权价和不同到期日。

  • Long Call Diagonal - 混合价差
  • Short Call Diagonal - 反向 call 对角
  • Long Put Diagonal - 看跌对角
  • Short Put Diagonal - 看涨对角

策略选择指南¶

如果您认为市场将...¶

大幅上涨 - Long Calls - Long Call Spread

大幅下跌 - Long Puts - Long Put Spread

保持平稳 - Short Straddle - Short Strangle - Iron Condor - Iron Butterfly

波动但方向不确定 - Long Straddle - Long Strangle

保持在当前价格附近 - Calendar Spreads - Butterflies

按风险/回报特征¶

有限风险,有限回报 - 所有价差(垂直、蝶式、iron condor)

有限风险,无限回报 - Long calls/puts - Long straddles/strangles

重大风险,有限回报 - Short calls/puts - Short straddles/strangles

通用参数¶

所有策略都接受这些通用参数:

  • max_entry_dte - 进场时的最大 DTE
  • exit_dte - 出场时的 DTE
  • dte_interval - 结果分组间隔
  • max_otm_pct - 最大虚值百分比
  • otm_pct_interval - OTM% 分组间隔
  • min_bid_ask - 最小买卖价差过滤器
  • raw - 返回原始交易数据(默认:聚合统计)

查看 参数指南 获取完整详细信息。
